Super PRD
==============

Super you PRD Style.

- Hide the checkbox in sidebar;
- Better font;
- Image shadow;
- More quote style
    - `> **spec**`
    - `> **info**`
    - `> **note**`
    - `> **hint**`
- `> **app**`, a new feature. 


## Usage
Add the plugin to your book.json like this:

```
{
    "plugins": ["super-prd"]
}
```
